# Security Policy

## Reporting a Vulnerability

Report vulnerabilities through GitHub private vulnerability reporting for this repository.

Do not publish exploit details, credentials, tokens, personal local paths, personal information, or private medical information in public issues, pull requests, comments, screenshots, or logs.

## Scope

This repository contains a fictional drug and disease mock server for application development. It is not a medical information source and must not be used for medical decisions, diagnosis, treatment, or clinical practice.
